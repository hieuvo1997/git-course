1. Giải thích các câu lệnh sau:

- git commit --amend: Mở editor để sửa message -> cái này tự search trong bài học không thấy ạ!
- git commit --amend -m"maessage": sửa massage của commit gần nhất
- git restored --staged <file>: restore 1 file từ staging về working directory
- git reset HEAD~1: reset 1 commit gần nhất
